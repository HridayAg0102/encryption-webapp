## Steps to run on local:
1. `pip install gradio`
2. run the python script `main.py`
3. open the link :

	```
	http://127.0.0.1:7860/
	```

4. While running the app on local, you'll get another (SHARABLE) 
	link where the app is hosted for 72 hours
